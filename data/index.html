URL Eater

Goals:

  • Add url text input
  • Add modifier text input
  • Add confirm button
  • Add output panel
  • Add copy-to-clipboard button
  • Add clear-all-text button
  • Add visit url button
  • Stylise all components

Instructions:

  • Input your URL to be modified in the 'URL' field.
  • Input your modifier text in the 'Modifier' field.
  • Place your caret at the desired point to insert the modified text.
  • If no caret placement is detected, the modified text will be added to the end of the input text by default.
  • Click the 'Modify URL!' button.
  • Done! You now have a modified URL.
  • Optionally click the 'Copy to Clipboard' button.
  • Optionally click the 'Visit URL' button.
